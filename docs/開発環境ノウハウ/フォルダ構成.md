## フォルダ構成

```
Meridian_TWIN_Tsy40
 │
 ├─ lib/
 │   ├ gs2d (汎用サーボライブラリ/順次対応)
 │   └ ICSClass_V210 (サーボ用ライブラリ)
 │
 ├─ src/
 │   ├ config.h      (設定ファイル)
 │   ├ main.h        (ヘッダファイル)
 │   ├ main.cpp      (Meridian本体)
 │   ├ mrd_disp.h    (以下各種コード)
 │   ├ mrd_eeprom.h
 │   ├ mrd_hwtimer.h
 │   ├ mrd_pad.h
 │   ├ mrd_sd.h
 │   ├ mrd_servo.h
 │   ├ mrd_wire0.h
 │   ├ mrd_wire1.h
 │   ├ mrd_util.h
 │   ├ mrd_move.h
 │   │
 │   └ mrd_module/
 │         └ (各種モジュール)
 │
 ├─ .clang-format (コード自動整形設定ファイル)
 └─ platfomio.ini (ボード設定ファイル)  </p>
```
